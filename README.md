# Структуры и алгоритмы обработки данных

Учебный проект на C: лабораторные работы по курсу «Структуры и алгоритмы обработки данных».
Каждая программа реализует один или несколько алгоритмов, выводит результаты в виде таблиц
и сравнивает фактические значения трудоёмкости (M — пересылки, C — сравнения)
с теоретическими оценками из методички.

## Сборка и запуск

### Требования
- `gcc` с поддержкой C99 (или новее)
- GNU Make
- Стандартная `libm` (математика)

### Сборка
```sh
make              # собрать все 15 программ
make <target>     # собрать одну программу (см. список ниже)
make clean        # удалить build/ и project.pack
make help         # показать встроенную справку
```

Бинарники появляются в `build/bin/`.

### Запуск
```sh
./build/bin/1_1_select_sort     # программа задачи 1.1
./build/bin/2_2_merge_sort      # программа задачи 2.2
./build/bin/2_4_hash_chain      # программа задачи 2.4
```
Программы не требуют параметров и сами выводят все таблицы.

Для удобства Makefile принимает и короткие алиасы: `make select`, `make merge`,
`make hash_chain` — собирают тот же бинарник, что и `make 1_1_select_sort`,
`make 2_2_merge_sort`, `make 2_4_hash_chain` соответственно.

### Сериализация исходников
Для удобной передачи проекта одним файлом:
```sh
make pack         # упаковать все .c/.h/Makefile/README в project.pack
make unpack       # распаковать обратно
```

## Теоретическая часть

Все теоретические оценки M и C взяты из таблицы [docs/labor_intensity.pdf](docs/labor_intensity.pdf).
Каждая программа выводит:

1. **Главную таблицу ТЗ** — по форме методички. Колонка «M+C теоретич.»
   расширена до трёх (Убыв./Случ./Возр.), если теория зависит от типа массива
   (Bubble, Shaker, Insert, Quick); одна колонка, если теория одинакова для всех
   случаев (Select, Heap, Merge, Digital).
2. **Раздел `=== Доп. ===`** — детальный разбор: теория `Mt`, `Ct` отдельно vs
   фактические `Мф`, `Сф` по типам массивов.

На убывающих и возрастающих массивах фактические значения **точно совпадают**
с теорией; на случайных — близки к среднему случаю (отклонение в пределах
нескольких процентов).
